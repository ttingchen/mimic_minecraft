# mimic_minecraft
![Imgur](https://i.imgur.com/BsT2BFk.jpg)

## Techniques
* Terrain generation
* Render model & Texture
* Skybox
* Normal mapping + Phong lighting
* Shadow Mapping
* SSAO
* Switch of every rendering technique

## Control
* mouse : trackball
* w/a/s/d : walk
* z/x : up/down
* n : normal mapping
* m : shadow
* j : ambient occlussion (still a bit weird)
